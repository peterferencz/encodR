# Shannon–Fano kódoló és dekódoló
BME VIK, Programozás 1, 1. Nagyházifeladat

### A program célja
Egy megadott bemeneti fájl kódolása Shannon–Fano kódolással, vagy annak dekódolása.

### Fejlesztés közben
```sh
hexdump [file] -C
xxd -b [file]
```
